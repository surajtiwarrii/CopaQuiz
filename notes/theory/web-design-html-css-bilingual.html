Web Design का परिचयIntroduction to Web Design

Website और Webpage क्या है?

What is a Website and Webpage?

Webpage एक single document होता है जो browser में दिखता है। कई related webpages को एक साथ जोड़कर Website बनती है, जो एक domain name (जैसे example.com) पर host होती है।

A Webpage is a single document displayed in a browser. Many related webpages linked together form a Website, hosted under one domain name (like example.com).

Static vs Dynamic Website

Static vs Dynamic Website

Static WebsiteDynamic Website
Content fixed रहता है, हर बार same दिखता हैContent stays fixed, shows the same every timeContent user/database के अनुसार बदलता हैContent changes based on user/database
HTML, CSSPHP, ASP.NET, Node.js + Database
Banane में आसान, सस्ताEasier and cheaper to buildज़्यादा powerful पर complexMore powerful but complex

Client-Server Model

Client-Server Model

जब हम browser में कोई URL खोलते हैं, browser (Client) request Server को भेजता है। Server उस webpage की files (HTML/CSS/JS) वापस भेजता है, जिसे browser render करके दिखाता है।

When we open a URL in a browser, the browser (Client) sends a request to the Server. The server sends back the webpage's files (HTML/CSS/JS), which the browser renders and displays.

HTML, CSS, JavaScript का Role

Role of HTML, CSS, JavaScript

TechnologyकामJob
HTMLWebpage की Structure/Content बनाता है (Skeleton)Builds the Structure/Content of the page (Skeleton)
CSSWebpage को style/design देता है (रंग, फॉन्ट, layout)Styles/designs the page (colors, fonts, layout)
JavaScriptWebpage को interactive/dynamic बनाता हैMakes the page interactive/dynamic
याद रखें: HTML = Structure, CSS = Style, JS = Behavior — ये तीनों मिलकर एक webpage बनाते हैं। Remember: HTML = Structure, CSS = Style, JS = Behavior — together they make a webpage.

HTML परिचय & StructureHTML Intro & Structure

HTML क्या है?

What is HTML?

HTML (HyperText Markup Language) एक markup language है जिससे webpages की structure बनाई जाती है। यह Tags का use करके content को define करता है — जैसे heading, paragraph, image, link आदि।

HTML (HyperText Markup Language) is a markup language used to build the structure of webpages. It uses Tags to define content — like headings, paragraphs, images, links, etc.

Hyperlink — <a> Tag

Hyperlink — <a> Tag

<a> tag से एक page से दूसरे page/website को link किया जाता है। href attribute destination देता है।

The <a> tag links one page to another page/website. The href attribute gives the destination.

<a href="https://example.com" target="_blank">Visit Site</a>
AttributeकामPurpose
hrefLink की destination URLDestination URL of the link
target="_blank"Link को नए tab में खोलता हैOpens the link in a new tab

Image — <img> Tag

Image — <img> Tag

<img> एक self-closing tag है, इसमें src (image path) और alt (image न दिखे तो text) ज़रूरी attributes हैं।

<img> is a self-closing tag, with src (image path) and alt (fallback text) as key attributes.

<img src="logo.png" alt="Logo" width="120">

Audio & Video Tags

The controls attribute provides a built-in play/pause/volume interface.

HTML5 Semantic TagsHTML5 Semantic Tags

Semantic Tags क्या हैं?

What are Semantic Tags?

Semantic Tags वो tags हैं जो सिर्फ structure ही नहीं देते बल्कि content का meaning भी बताते हैं — जैसे <div> की जगह <header> use करना browser और search engine दोनों को समझाता है कि यह header वाला हिस्सा है।

Semantic Tags are tags that not only give structure but also convey the meaning of the content — using <header> instead of <div> tells both the browser and search engines that this is a header section.

Common Semantic Tags

<footer>Page/section का bottom हिस्सा — copyright, contactBottom part of a page/section — copyright, contact
<header>...</header> <nav>...</nav> <main> <section> <article>...</article> </section> </main> <footer>...</footer>

iframe Tag

iframe Tag

<iframe> एक webpage के अंदर दूसरे webpage को embed करने के लिए use होता है — जैसे YouTube video या Google Map embed करना।

<iframe> is used to embed another webpage inside the current webpage — like embedding a YouTube video or Google Map.

<iframe src="page.html" width="300" height="200"></iframe>
याद रखें: Semantic tags use करने से SEO अच्छा होता है और code पढ़ने में आसान होता है (readability)। Remember: using semantic tags improves SEO and makes code easier to read (readability).

CSS परिचय & SelectorsCSS Intro & Selectors

CSS क्या है?

What is CSS?

CSS (Cascading Style Sheets) HTML elements को style करने के लिए use होता है — रंग, फॉन्ट, spacing, layout आदि। "Cascading" का मतलब है rules ऊपर से नीचे priority के हिसाब से apply होते हैं।

CSS (Cascading Style Sheets) is used to style HTML elements — colors, fonts, spacing, layout, etc. "Cascading" means rules apply top-to-bottom based on priority.

CSS लगाने के तीन तरीके

Three Ways to Apply CSS

TypeकामDescription
Inline CSSTag के अंदर ही style attribute सेDirectly inside the tag using the style attribute
Internal CSS<head> में <style> tag के अंदरInside a <style> tag within <head>
External CSSअलग .css file, <link> tag से जोड़ा जाता हैA separate .css file, linked via the <link> tag
<!-- Inline --> <p style="color:blue;">Text</p> <!-- Internal --> <style> p { color: blue; } </style> <!-- External --> <link rel="stylesheet" href="style.css">

CSS Selectors

.classकिसी particular class वाले elements को select करता हैSelects elements with a particular class

*सभी elements को select करता है (Universal selector)Selects all elements (Universal selector)
.highlight { background: yellow; } #header { font-size: 22px; } p { line-height: 1.6; }
याद रखें: ID पूरे page में सिर्फ एक बार use होनी चाहिए, जबकि class को कई elements पर repeat किया जा सकता है। Remember: an ID should be used only once per page, while a class can be reused on multiple elements.

Every HTML element is a box made of 4 parts, from inside out:

Linear Gradient

Web Safe Fonts & Google Fonts

Web Safe Fonts & Google Fonts

Web Safe Fonts वो fonts हैं जो लगभग हर device/OS में pre-installed होते हैं (जैसे Arial, Times New Roman)। Google Fonts से custom fonts <link> tag के through page में import किए जा सकते हैं।

Web Safe Fonts are fonts pre-installed on almost every device/OS (like Arial, Times New Roman). Google Fonts let you import custom fonts into a page via the <link> tag.

याद रखें: font-family में हमेशा एक fallback generic font (जैसे sans-serif) दिया जाता है, अगर पहला font load न हो तो। Remember: font-family should always include a fallback generic font (like sans-serif) in case the first font fails to load.

fixedBrowser window के relative fixed रहता है, scroll पर भी नहीं हिलताFixed relative to the browser window, doesn't move on scroll
stickyNormal flow में रहता है, पर scroll होने पर एक point पर stick हो जाता हैStays in normal flow but sticks at a point when scrolling
.box { position: absolute; top: 20px; left: 30px; }

top, right, bottom, left

top, right, bottom, left

ये properties सिर्फ तभी काम करती हैं जब position को static के अलावा कुछ और set किया गया हो — ये बताती हैं element कितना दूर offset होगा।

These properties only work when position is set to something other than static — they specify how far the element is offset.

z-index

z-index

z-index बताता है कि overlapping elements में से कौन सा element ऊपर (front में) दिखेगा। ज़्यादा value वाला element ऊपर आता है।

z-index determines which element appears on top (front) among overlapping elements. A higher value comes on top.

Exam Tip: Sticky vs Fixed में confusion होता है — Sticky scroll के एक point तक normal रहता है, फिर stick होता है; Fixed शुरू से viewport के relative fixed रहता है। Exam Tip: Sticky vs Fixed often confuses students — Sticky behaves normally until a scroll point, then sticks; Fixed stays fixed to the viewport from the start.

float Property

float Property

float किसी element को left या right में push करता है, बाकी content उसके चारों ओर wrap हो जाता है — पुराने layouts में बहुत use होता था।

float pushes an element to the left or right, with other content wrapping around it — heavily used in older layouts.

img { float: left; margin-right: 10px; } .clearfix { clear: both; }

clear property बताती है कि कोई element float हुए elements के बगल में नहीं बल्कि नीचे आएगा।

The clear property specifies that an element should move below floated elements instead of beside them.

Pseudo-classes & Pseudo-elementsPseudo-classes & Pseudo-elements

Pseudo-class क्या है?

What is a Pseudo-class?

Pseudo-class किसी element की special state को target करता है, जैसे hover किया जाना या कोई particular position पर होना। यह single colon (:) से लिखा जाता है।

A Pseudo-class targets a special state of an element, like being hovered or being at a particular position. It's written with a single colon (:).

Pseudo-classकामPurpose
:hoverMouse element के ऊपर आने पर style apply करता हैApplies style when mouse hovers over element
:focusElement (जैसे input) focus होने पर style apply करता हैApplies style when element (e.g. input) is focused
:first-childParent के पहले child को select करता हैSelects the first child of the parent
:last-childParent के last child को select करता हैSelects the last child of the parent
:nth-child(n)n-वें child को select करता हैSelects the nth child
a:hover { color: #0369a1; } input:focus { border-color: #38bdf8; }

Pseudo-element क्या है?

What is a Pseudo-element?

Pseudo-element किसी element के specific हिस्से को style करता है, जैसे पहली line या कोई virtual content add करना। यह double colon (::) से लिखा जाता है।

A Pseudo-element styles a specific part of an element, like the first line, or adds virtual content. It's written with a double colon (::).

Pseudo-elementकामPurpose
::beforeElement के content से पहले virtual content add करता हैAdds virtual content before the element's content
::afterElement के content के बाद virtual content add करता हैAdds virtual content after the element's content
::first-letterText के पहले letter को style करता हैStyles the first letter of text
.unit-body h3::before { content: ""; background: #38bdf8; }
याद रखें: Pseudo-class state target करता है (single colon :), Pseudo-element हिस्सा target करता है (double colon ::)। Remember: Pseudo-class targets a state (single colon :), Pseudo-element targets a part (double colon ::).

Responsive Web DesignResponsive Web Design

Responsive Design क्या है?

What is Responsive Design?

Responsive Web Design ऐसी design approach है जिसमें webpage अलग-अलग screen sizes (mobile, tablet, desktop) पर अच्छी तरह adjust हो जाता है, बिना अलग-अलग version बनाए।

Responsive Web Design is a design approach where a webpage adapts well to different screen sizes (mobile, tablet, desktop) without building separate versions.

Viewport Meta Tag

Viewport Meta Tag

Mobile devices पर सही scaling के लिए यह tag <head> में ज़रूर डालना चाहिए:

This tag must be included in <head> for correct scaling on mobile devices:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

Media Queries

Media Query CSS का एक feature है जो screen size के according अलग-अलग styles apply करने देता है — @media rule से लिखा जाता है।

A Media Query is a CSS feature that applies different styles based on screen size — written using the @media rule.

@media (max-width: 600px) { body { font-size: 14px; } .layout { display: block; } }

Mobile-First vs Desktop-First

Mobile-First vs Desktop-First

Mobile-FirstDesktop-First
पहले mobile के लिए design करते हैं, फिर बड़ी screens के लिए बढ़ाते हैंDesign for mobile first, then scale up for larger screensपहले desktop के लिए design करते हैं, फिर छोटी screens के लिए adjust करते हैंDesign for desktop first, then adjust for smaller screens
Exam Tip: Responsive Design के तीन मुख्य elements हैं — Flexible Grid/Layout, Flexible Images, और Media Queries। Exam Tip: the three main elements of Responsive Design are Flexible Grid/Layout, Flexible Images, and Media Queries.
