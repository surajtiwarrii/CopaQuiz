Python का परिचयIntroduction to Python

Python क्या है?

What is Python?

Python एक high-level, interpreted, general-purpose programming language है जिसे Guido van Rossum ने 1991 में develop किया था। यह अपनी simple, readable syntax के लिए जाना जाता है — beginners के लिए best language मानी जाती है।

Python is a high-level, interpreted, general-purpose programming language developed by Guido van Rossum in 1991. Known for its simple, readable syntax — considered the best language for beginners.

Variables & Data Types

Variable क्या है?

What is a Variable?

Python में variable declare करने की ज़रूरत नहीं — सिर्फ value assign करो, automatically variable create हो जाता है। Type भी automatically decide होता है (Dynamic Typing)।

No need to declare a variable in Python — just assign a value and it is automatically created. The type is also decided automatically (Dynamic Typing).

x = 10 # int y = 3.14 # float name = "Suraj" # str flag = True # bool

Strings & String Methods

String क्या है?

What is a String?

String characters का sequence है, single (') या double (") quotes में लिखी जाती है। Python में String immutable है — create होने के बाद change नहीं होती।

A String is a sequence of characters, written in single (') or double (") quotes. Strings in Python are immutable — they cannot be changed after creation.

s = "Hello Python" print(s[0]) # 'H' (indexing) print(s[6:]) # 'Python' (slicing) print(s[:5]) # 'Hello' print(s[::-1]) # Reverse string print(len(s)) # 12

Lists

List क्या है?

What is a List?

List ordered, mutable collection है जो अलग-अलग data types की values store कर सकती है। Square brackets [ ] में लिखी जाती है। Index 0 से शुरू होती है।

A List is an ordered, mutable collection that can store values of different data types. Written in square brackets [ ]. Indexing starts from 0.

fruits = ["Apple", "Banana", "Mango"] nums = [1, 2, 3, 4, 5] mixed = [1, "hello", 3.14, True] print(fruits[0]) # Apple print(fruits[-1]) # Mango (negative index) print(fruits[1:3]) # ['Banana', 'Mango']

Tuples, Dictionaries & Sets

Tuple

Tuple

Tuple ordered, immutable collection है — एक बार create होने के बाद change नहीं हो सकती। Parentheses () में लिखी जाती है।

A Tuple is an ordered, immutable collection — cannot be changed after creation. Written in parentheses ().

t = (1, 2, 3, "hello") print(t[0]) # 1 print(len(t)) # 4 # t[0] = 5 → Error! (immutable)

Dictionary

Dictionary

Dictionary key-value pairs में data store करता है। Curly braces {} में, colon से key और value separate। Keys unique होती हैं।

A Dictionary stores data in key-value pairs. Written in curly braces {} with a colon separating key and value. Keys must be unique.

student = { "name": "Suraj", "age": 21, "course": "ITI-COPA" } print(student["name"]) # Suraj print(student.get("age")) # 21 student["city"] = "Gonda" # New key add print(student.keys()) # all keys print(student.values()) # all values

Set

Set

Set unordered collection है जिसमें duplicate values allowed नहीं हैं। Curly braces {} में लिखा जाता है।

A Set is an unordered collection in which duplicate values are not allowed. Written in curly braces {}.

s = {1, 2, 3, 2, 1} print(s) # {1, 2, 3} (duplicates removed)

Functions

Function क्या है?

What is a Function?

Function code का reusable block है — def keyword से define होती है।

A Function is a reusable block of code — defined using the def keyword.

def greet(name): print(f"Hello, {name}!") greet("Suraj") # Hello, Suraj! def add(a, b): return a + b result = add(5, 3) print(result) # 8

Default Arguments

Default Arguments

def greet(name, msg="Hello"): # default value print(msg, name) greet("Suraj") # Hello Suraj greet("Suraj", "Hi") # Hi Suraj

Lambda Function

Lambda Function

Lambda anonymous (nameless) function है — single line में लिखी जाती है।

A Lambda is an anonymous (nameless) function — written in a single line.

square = lambda x: x * x print(square(5)) # 25 add = lambda a, b: a + b print(add(3, 4)) # 7

*args & **kwargs

*args & **kwargs

def total(*args): # multiple positional args return sum(args) print(total(1,2,3,4)) # 10

File Handling

File Handling क्यों?

Why File Handling?

File Handling से Python program data को files में store (write) करके दोबारा read कर सकता है — permanent data storage के लिए।

File Handling allows a Python program to store (write) data in files and read it again — for permanent data storage.

Exception Handling & Modules

Exception क्या है?

What is an Exception?

Exception program execution के दौरान होने वाली error है जो program को crash कर देती है। Exception Handling से program gracefully error handle करता है।

An Exception is an error during program execution that causes the program to crash. Exception Handling allows the program to handle errors gracefully.

try: x = int(input("Enter number: ")) result = 10 / x print(result) except ZeroDivisionError: print("Cannot divide by zero!") except ValueError: print("Please enter a valid number!") finally: print("This always runs.")

Modules

Modules

Module Python code का एक reusable file है। import keyword से module को use किया जाता है।

A Module is a reusable Python code file. The import keyword is used to use a module.

import math print(math.sqrt(25)) # 5.0 print(math.pi) # 3.14159... import random print(random.randint(1, 10)) # Random number 1-10 from datetime import date print(date.today()) # Today's date
