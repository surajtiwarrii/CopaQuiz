Booting क्या है?

What is Booting?

Booting वो process है जिसके through computer को ON करने पर वह start होता है — Operating System को storage (Hard Disk) से RAM में load करना और system को use के लिए तैयार करना।

Booting is the process by which a computer starts when switched on — loading the Operating System from storage (Hard Disk) into RAM and making the system ready for use.

यह नाम "Bootstrapping" से आया है — जैसे कोई खुद को जूते के फीतों से उठाए, वैसे computer खुद को start करता है।

The name comes from "Bootstrapping" — like someone pulling themselves up by their bootstraps, the computer starts itself up.

POST (Power On Self Test)

POST क्या है?

What is POST?

POST (Power On Self Test) computer ON होने पर सबसे पहले run होने वाला diagnostic program है — यह hardware components को check करता है ताकि सुनिश्चित हो सके कि सब कुछ properly काम कर रहा है।

POST (Power On Self Test) is the first diagnostic program that runs when the computer is switched on — it checks hardware components to ensure everything is working properly.

POST किस Hardware को Check करता है?

What Hardware does POST Check?

  • CPU (Processor) — working है या नहीं
  • RAM — installed और functional है या नहीं
  • ROM/BIOS chip — readable है या नहीं
  • Keyboard — connected है या नहीं
  • Video Card / Display — working है या नहीं
  • Storage devices (HDD/SSD) — detected हो रहे हैं या नहीं
  • CPU (Processor) — whether it is working
  • RAM — whether it is installed and functional
  • ROM/BIOS chip — whether it is readable
  • Keyboard — whether it is connected
  • Video Card / Display — whether it is working
  • Storage devices (HDD/SSD) — whether they are detected

POST के Results

POST Results

No Beep / 1 BeepPOST successful — hardware सब ठीक है, booting continue होगीPOST successful — hardware is fine, booting will continue
Multiple BeepsPOST failed — hardware में problem है, beep codes से problem identify होती हैPOST failed — there is a hardware problem; beep codes identify the issue
याद रखें: POST BIOS द्वारा run किया जाता है — यह OS load होने से पहले का पहला step है।Remember: POST is run by the BIOS — it is the very first step before the OS loads.

BIOS & UEFI

BIOS क्या है?

What is BIOS?

BIOS (Basic Input Output System) Motherboard की ROM chip पर store firmware है — यह computer ON होने पर सबसे पहले load होता है, POST run करता है, hardware initialize करता है, और OS को load करने की chain शुरू करता है।

BIOS (Basic Input Output System) is firmware stored on a ROM chip on the Motherboard — it loads first when the computer starts, runs POST, initialises hardware, and begins the chain to load the OS.

BIOS के काम

Functions of BIOS

  • POST run करना
  • Hardware को initialize करना
  • Boot device ढूँढना
  • OS को RAM में load करना शुरू करना
  • Hardware settings (date/time, boot order) store करना
  • Running POST
  • Initialising hardware
  • Finding the boot device
  • Starting to load the OS into RAM
  • Storing hardware settings (date/time, boot order)

BIOS vs UEFI

BIOS vs UEFI

BIOSUEFI (Unified Extensible Firmware Interface)

Max 2TB hard disk supportMax 2TB hard disk support2TB से बड़े drives support करता हैSupports drives larger than 2TB
Slow bootSlow bootFaster boot — Secure Boot featureFaster boot — Secure Boot feature

BIOS Setup में जाना

Accessing BIOS Setup

Computer start होने पर specific key press करने से BIOS setup खुलता है — common keys हैं: Del, F2, F10, F12 (manufacturer के हिसाब से अलग होती है)।

Press a specific key as the computer starts to enter BIOS setup — common keys are Del, F2, F10, F12 (varies by manufacturer).

Boot Sequence & MBRBoot Sequence & MBR

Boot Sequence (Boot Order) क्या है?

What is Boot Sequence (Boot Order)?

Boot Sequence वो order है जिसमें BIOS bootable OS ढूँढने के लिए storage devices को check करता है — जैसे पहले HDD, फिर USB, फिर CD/DVD।

Boot Sequence is the order in which BIOS checks storage devices to find a bootable OS — e.g. first HDD, then USB, then CD/DVD.

BIOS setup में boot order change किया जा सकता है — जैसे USB से boot करने के लिए USB को पहले रखते हैं।

Boot order can be changed in BIOS setup — e.g. to boot from USB, USB is placed first.

MBR (Master Boot Record)

MBR (Master Boot Record)

MBR Hard Disk के पहले sector (512 bytes) में stored एक special area है जिसमें:

MBR is a special area stored in the first sector (512 bytes) of the Hard Disk containing:

  • Boot Loader — OS को load करने वाला small program
  • Partition Table — disk के partitions (C:, D:) की जानकारी
  • Boot Loader — a small program that loads the OS
  • Partition Table — information about disk partitions (C:, D:)

Boot Loader क्या है?

What is a Boot Loader?

Boot Loader एक small program है जो OS को RAM में load करता है — Windows में यह bootmgr (Boot Manager) होता है, Linux में GRUB होता है।

A Boot Loader is a small program that loads the OS into RAM — in Windows it is bootmgr (Boot Manager), in Linux it is GRUB.

याद रखें: BIOS → MBR → Boot Loader → OS Kernel Load — यह basic chain है।Remember: BIOS → MBR → Boot Loader → OS Kernel Load — this is the basic chain.

CMOS & BIOS SettingsCMOS & BIOS Settings

CMOS क्या है?

What is CMOS?

CMOS (Complementary Metal Oxide Semiconductor) Motherboard पर एक small memory chip है जो BIOS settings, date/time, और hardware configuration store करती है — यह एक small button cell battery से powered रहती है।

CMOS (Complementary Metal Oxide Semiconductor) is a small memory chip on the Motherboard that stores BIOS settings, date/time, and hardware configuration — it is powered by a small button cell battery.

CMOS Battery

CMOS Battery

CMOS battery (CR2032) computer OFF होने पर भी settings को save रखती है। Battery dead होने पर date/time reset हो जाता है और BIOS settings lost हो जाती हैं।

The CMOS battery (CR2032) keeps settings saved even when the computer is off. When the battery dies, date/time resets and BIOS settings are lost.

Safe Mode & Boot ErrorsSafe Mode & Boot Errors

Safe Mode क्या है?

What is Safe Mode?

Safe Mode Windows का एक diagnostic startup mode है जिसमें only essential drivers और services load होते हैं — virus, driver problems, या software crashes के बाद troubleshoot करने के लिए use होता है।

Safe Mode is a Windows diagnostic startup mode in which only essential drivers and services load — used to troubleshoot after a virus, driver problems, or software crashes.

Safe Mode में क्या load होता हैWhat loads in Safe Modeक्या load नहीं होताWhat doesn't load
Basic drivers (keyboard, mouse, display)Basic drivers (keyboard, mouse, display)Third-party software, extra services, network (normally)Third-party software, extra services, network (normally)

Safe Mode में जाना: Computer start होने पर F8 press करें (Windows 7 में) या Settings → Update & Security → Recovery → Advanced Startup (Windows 10/11 में)।

To enter Safe Mode: Press F8 at startup (Windows 7) or Settings → Update & Security → Recovery → Advanced Startup (Windows 10/11).

Dual Booting क्या है?

What is Dual Booting?

Dual Boot एक computer पर दो OS install करना है — startup पर user choose करता है कौन सा OS boot हो। जैसे Windows + Linux एक साथ।

Dual Boot means installing two OS on one computer — at startup, the user chooses which OS to boot. E.g. Windows + Linux together.

Fast Startup vs Normal Shutdown

Fast Startup vs Normal Shutdown

Fast Startup (Windows 8+) Hybrid sleep mode use करता है — System session RAM में नहीं, hibernate file में save होती है, पर user data नहीं। इससे next boot बहुत fast होती है।

Fast Startup (Windows 8+) uses hybrid sleep — the system session is saved in a hibernate file (not user data). This makes the next boot much faster.
