MS Excel क्या है?

What is MS Excel?

MS Excel Microsoft का एक spreadsheet software है, जिसमें data को rows और columns की grid में organize, calculate, और analyze किया जाता है।

MS Excel is a spreadsheet software by Microsoft, used to organize, calculate, and analyze data in a grid of rows and columns.

Cell Address व Name Box

Cell Address & Name Box

हर cell का एक unique address होता है — Column letter + Row number (जैसे B5)। Name Box में address type करके सीधे उस cell पर जा सकते हैं।

Every cell has a unique address — Column letter + Row number (like B5). Typing an address in the Name Box jumps directly to that cell.

याद रखें: एक Workbook में by default कई Worksheets हो सकती हैं, और हर worksheet में 16,384 columns (A to XFD) और 1,048,576 rows होती हैं (Excel 2016+ में)। Remember: a Workbook can contain multiple Worksheets, and each worksheet has 16,384 columns (A to XFD) and 1,048,576 rows (in Excel 2016+).

Cell ReferencingCell Referencing

Cell Reference क्या है?

What is a Cell Reference?

Cell Reference किसी formula में किसी cell का address use करना है, ताकि उस cell की value formula में automatically आ जाए — manual typing की ज़रूरत नहीं पड़ती।

A Cell Reference means using a cell's address in a formula so its value is pulled in automatically — no need to type it manually.

Relative, Absolute & Mixed Reference

Relative, Absolute & Mixed Reference

TypeExampleविशेषताFeature
RelativeA1Copy करने पर row/column के हिसाब से बदल जाता हैChanges based on row/column when copied
Absolute$A$1Copy करने पर भी fix रहता है, $ sign लगाया जाता हैStays fixed when copied, marked with $ sign
Mixed$A1 या A$1सिर्फ column या सिर्फ row fix रहता हैOnly the column or only the row stays fixed
=A1*$B$1 // A1 relative, $B$1 absolute

F4 Key Shortcut

F4 Key Shortcut

Formula में किसी cell reference को select करके F4 press करने से Relative → Absolute → Mixed → वापस Relative cycle में बदलता है।

Selecting a cell reference in a formula and pressing F4 cycles it through Relative → Absolute → Mixed → back to Relative.

3D Reference

3D Reference

3D Reference कई worksheets की same cell range को एक formula में एक साथ refer करता है, जैसे =SUM(Sheet1:Sheet3!A1)।

A 3D Reference refers to the same cell range across multiple worksheets in one formula, like =SUM(Sheet1:Sheet3!A1).

याद रखें: Absolute reference में $ row और column दोनों के पहले लगता है (जैसे $A$1) ताकि copy/fill करने पर भी value fix रहे। Remember: in an absolute reference, $ appears before both the column and row (like $A$1) so the value stays fixed even when copied/filled.

Logical FunctionsLogical Functions

IF Function

IF Function

IF किसी condition को check करके दो में से एक result देता है — true होने पर एक value, false होने पर दूसरी।

IF checks a condition and returns one of two results — one value if true, another if false.

=IF(A1>=40,"Pass","Fail")

Nested IF

Nested IF

जब एक से ज़्यादा conditions check करनी हों, तब एक IF के अंदर दूसरा IF लगाया जाता है (Nested IF)।

When multiple conditions need checking, an IF is placed inside another IF (Nested IF).

=IF(A1>=90,"A",IF(A1>=75,"B",IF(A1>=60,"C","D")))

AND, OR, NOT Functions

AND, OR, NOT Functions

FunctionकामPurpose
AND()सभी conditions true हों तभी TRUE देता हैReturns TRUE only if all conditions are true
OR()कोई एक भी condition true हो तो TRUE देता हैReturns TRUE if any one condition is true
NOT()Condition को उल्टा कर देता हैReverses the condition
=IF(AND(A1>=40,B1>=40),"Pass","Fail")

IFERROR Function

IFERROR Function

IFERROR किसी formula में error आने पर एक custom message/value दिखाता है, ताकि #DIV/0! जैसी ugly errors छुप जाएं।

IFERROR shows a custom message/value when a formula returns an error, hiding ugly errors like #DIV/0!.

=IFERROR(A1/B1,"Error")
याद रखें: IF function के 3 arguments होते हैं — condition, value_if_true, value_if_false। Remember: the IF function has 3 arguments — condition, value_if_true, value_if_false.

Date Arithmetic

Date Arithmetic

Excel में dates को internally serial numbers की तरह store किया जाता है, इसलिए dates पर सीधे addition/subtraction किया जा सकता है — जैसे =A1+30 30 दिन बाद की date देगा।

Excel internally stores dates as serial numbers, so dates can be added/subtracted directly — e.g. =A1+30 gives the date 30 days later.

याद रखें: TODAY() और NOW() dynamic functions हैं — हर बार sheet open/calculate होने पर current date/time से update हो जाते हैं। Remember: TODAY() and NOW() are dynamic functions — they update to the current date/time every time the sheet opens/recalculates.

Lookup FunctionsLookup Functions

VLOOKUP Function

VLOOKUP Function

VLOOKUP (Vertical Lookup) किसी value को table के पहले column में ढूँढता है, और उसी row से किसी और column की value return करता है।

VLOOKUP (Vertical Lookup) searches for a value in the first column of a table and returns a value from another column in the same row.

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup]) =VLOOKUP(A2,Sheet2!A:C,3,FALSE)
ArgumentमतलबMeaning
lookup_valueजो value ढूँढनी हैThe value to search for
table_arrayजिस range में ढूँढना हैThe range to search within
col_index_numकिस column से value लानी हैWhich column to return the value from
range_lookupFALSE = exact match, TRUE = approximate matchFALSE = exact match, TRUE = approximate match

HLOOKUP Function

HLOOKUP Function

HLOOKUP (Horizontal Lookup) VLOOKUP जैसा ही है, पर data को rows में horizontally search करता है, columns में नहीं।

HLOOKUP (Horizontal Lookup) works like VLOOKUP but searches data horizontally across rows instead of columns.

INDEX & MATCH

INDEX & MATCH

INDEX-MATCH combination VLOOKUP से ज़्यादा flexible है — left की तरफ भी lookup कर सकता है और column add/remove होने पर भी सही काम करता है।

The INDEX-MATCH combination is more flexible than VLOOKUP — it can look up to the left and keeps working correctly even if columns are added/removed.

=INDEX(C:C,MATCH(A2,A:A,0))
Exam Tip: VLOOKUP में lookup column हमेशा table_array के सबसे left column में होना चाहिए — यही इसकी सबसे बड़ी limitation है। Exam Tip: in VLOOKUP, the lookup column must always be the leftmost column of the table_array — this is its biggest limitation.

Sorting & FilteringSorting & Filtering

Sorting क्या है?

What is Sorting?

Sorting data को एक particular order (Ascending/Descending) में arrange करता है — Numbers, Text, या Dates के according।

Sorting arranges data in a particular order (Ascending/Descending) — by Numbers, Text, or Dates.

TypeमतलबMeaning

Descending (Z-A)बड़े से छोटे / Z से A order मेंLargest to smallest / Z to A order

Filtering क्या है?

What is Filtering?

Filter data में से सिर्फ वही rows दिखाता है जो किसी condition को match करती हैं, बाकी temporarily hide हो जाती हैं।

A Filter shows only the rows matching a condition, hiding the rest temporarily.

AutoFilter vs Advanced Filter

याद रखें: Sort permanently data का order बदल देता है, जबकि Filter सिर्फ rows को temporarily hide करता है — data delete नहीं होता। Remember: Sort permanently changes the order of data, while Filter only hides rows temporarily — no data is deleted.

Conditional Formatting क्या है?

What is Conditional Formatting?

Conditional Formatting cells को उनकी value के according automatically format (रंग, icon, data bar) करता है — पैटर्न और trends को visually highlight करने के लिए।

Conditional Formatting automatically formats cells (color, icon, data bar) based on their value — useful for visually highlighting patterns and trends.

Data Validation क्या है?

What is Data Validation?

Data Validation control करता है कि किसी cell में कौन सा data enter किया जा सकता है — गलत/invalid entries रोकने के लिए (जैसे dropdown list, number range)।

Data Validation controls what data can be entered in a cell — preventing incorrect/invalid entries (like a dropdown list, number range).

A PivotTable is a powerful tool to quickly summarize, group, and analyze large data — without writing any formula, just by drag-and-drop.

What-If AnalysisWhat-If Analysis

What-If Analysis क्या है?

What is What-If Analysis?

What-If Analysis tools की मदद से अलग-अलग values डालकर देखा जाता है कि formula के results कैसे बदलते हैं — बिना actual data permanently change किए।

What-If Analysis tools let you test different values to see how formula results change — without permanently altering the actual data.

Goal Seek

Goal Seek

Goal Seek उल्टा तरीके से काम करता है — आप desired result बताते हैं, और Excel calculate करता है कि input value क्या होनी चाहिए।

Goal Seek works backwards — you specify the desired result, and Excel calculates what the input value should be.

FieldFieldमतलबMeaning
Set Cellवो formula cell जिसका result चाहिएThe formula cell whose result you want
To Valueवो target value जो चाहिएThe target value you want
By Changing Cellवो input cell जो adjust होगीThe input cell that will be adjusted

Data Table

Data Table

Data Table एक formula पर कई अलग-अलग input values का effect एक साथ table में देखने देता है (One-Variable या Two-Variable Data Table)।

A Data Table shows the effect of multiple different input values on a formula at once, in a table (One-Variable or Two-Variable Data Table).

Scenario Manager

Scenario Manager

Scenario Manager कई अलग-अलग input combinations ("scenarios") को save करके, switch करके compare करने देता है — जैसे Best Case, Worst Case, Average Case।

Scenario Manager lets you save multiple different input combinations ("scenarios") and switch between them to compare — like Best Case, Worst Case, Average Case.

याद रखें: Goal Seek सिर्फ एक input variable adjust करता है, जबकि Scenario Manager कई variables के combination save कर सकता है। Remember: Goal Seek adjusts only one input variable, while Scenario Manager can save combinations of multiple variables.

Macro क्या है?

What is a Macro?

Macro actions की एक recorded series है जिसे एक click में बार-बार चलाया जा सकता है — repetitive tasks को automate करने के लिए use होता है।

A Macro is a recorded series of actions that can be replayed with a single click — used to automate repetitive tasks.

Macro Record करना

Recording a Macro

  1. View tab → Macros → Record Macro
  2. Macro को नाम दें और shortcut key (optional) set करें
  3. जो actions automate करनी हैं वो perform करें
  4. View tab → Macros → Stop Recording
  1. View tab → Macros → Record Macro
  2. Give the macro a name and an optional shortcut key
  3. Perform the actions you want to automate
  4. View tab → Macros → Stop Recording

VBA (Visual Basic for Applications)

VBA (Visual Basic for Applications)

VBA वो programming language है जिसमें Excel macros internally लिखे जाते हैं — Developer tab → Visual Basic से VBA editor खोला जा सकता है।

VBA is the programming language in which Excel macros are written internally — the VBA editor can be opened via Developer tab → Visual Basic.
