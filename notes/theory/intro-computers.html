Computer का परिचयIntroduction to Computer

Computer क्या है?

What is a Computer?

Computer एक electronic machine है जो data को input के रूप में लेता है, उसे process करता है, और output के रूप में result देता है — साथ ही data को store भी कर सकता है।

A Computer is an electronic machine that takes data as input, processes it, and gives the result as output — it can also store data.

IPO Cycle

IPO Cycle

Input → Process → Output — यह computer का basic working model है। Data → CPU (processing) → Result।

Input → Process → Output — this is the basic working model of a computer. Data → CPU (processing) → Result.

Limitations of Computer

Limitations of Computer

  • खुद से सोच नहीं सकता — सिर्फ दिए गए instructions follow करता है
  • Common sense नहीं होती
  • Electricity पर depend करता है
  • Virus/malware से affect हो सकता है
  • Cannot think on its own — only follows given instructions
  • Has no common sense
  • Dependent on electricity
  • Can be affected by viruses/malware
याद रखें: Computer की full form कोई official नहीं है, पर एक common expansion है: Common Operating Machine Purposely Used for Technological and Educational Research।Remember: Computer has no official full form, but one common expansion is: Common Operating Machine Purposely Used for Technological and Educational Research.

CPU क्या है?

What is the CPU?

CPU (Central Processing Unit) computer का brain है — यह सभी instructions को process करता है और सभी operations को control करता है।

The CPU (Central Processing Unit) is the brain of the computer — it processes all instructions and controls all operations.

Cache Memory

1-line Definitions

1-line Definitions

  • Computer — Electronic machine जो data input लेकर process करके output देता है
  • CPU — Computer का brain, सभी operations process और control करता है
  • RAM — Volatile, read-write primary memory जो current data store करती है
  • ROM — Non-volatile, read-only memory जो BIOS store करती है
  • Input Device — User से data computer में enter करवाने वाला device
  • Output Device — Computer का result user को display करने वाला device
  • Hardware — Computer के physical parts जिन्हें touch किया जा सकता है
  • Software — Programs और instructions जो hardware को control करते हैं
  • Binary — Computer की language जिसमें सिर्फ 0 और 1 होते हैं
  • Cache Memory — CPU के अंदर fastest memory
  • Computer — Electronic machine that takes data input, processes it, and gives output
  • CPU — Brain of the computer, processes and controls all operations
  • RAM — Volatile, read-write primary memory that stores current data
  • ROM — Non-volatile, read-only memory that stores BIOS
  • Input Device — Device that allows users to enter data into the computer
  • Output Device — Device that displays the computer's result to the user
  • Hardware — Physical parts of a computer that can be touched
  • Software — Programs and instructions that control hardware
  • Binary — Computer's language consisting only of 0 and 1
  • Cache Memory — The fastest memory inside the CPU
