DOS परिचयDOS Introduction

DOS क्या है?

What is DOS?

DOS (Disk Operating System) एक text-based, single-user, single-tasking operating system है जो 1980s-90s में PCs पर use होता था। Microsoft का version MS-DOS सबसे popular था।

DOS (Disk Operating System) is a text-based, single-user, single-tasking operating system used on PCs in the 1980s-90s. Microsoft's version, MS-DOS, was the most popular.

DOS की विशेषताएं

Features of DOS

  • Command Line Interface (CLI) — commands type करके use करना पड़ता है
  • Single user — एक समय पर एक user
  • Single tasking — एक समय पर एक task
  • No graphics — pure text interface
  • Case insensitive — commands uppercase या lowercase दोनों में काम करते हैं
  • Command Line Interface (CLI) — must type commands to use
  • Single user — one user at a time
  • Single tasking — one task at a time
  • No graphics — pure text interface
  • Case insensitive — commands work in uppercase or lowercase

DOS File Naming Rules (8.3 Format)

DOS File Naming Rules (8.3 Format)

  • Filename maximum 8 characters, extension maximum 3 characters
  • Extension dot (.) से separate होती है — जैसे FILE.TXT
  • Spaces allowed नहीं हैं filename में
  • Special characters (/ \ : * ? " < > |) allowed नहीं
  • Case insensitive — FILE.TXT और file.txt same हैं
  • Filename maximum 8 characters, extension maximum 3 characters
  • Extension is separated by a dot (.) — e.g. FILE.TXT
  • Spaces are not allowed in filenames
  • Special characters (/ \ : * ? " < > |) are not allowed
  • Case insensitive — FILE.TXT and file.txt are the same

Linux क्या है?

What is Linux?

Linux एक open-source, Unix-like operating system है जिसे Linus Torvalds ने 1991 में develop किया था। यह free में available है और दुनिया के most servers, smartphones (Android), और supercomputers पर run होता है।

Linux is an open-source, Unix-like operating system developed by Linus Torvalds in 1991. It is freely available and runs on most of the world's servers, smartphones (Android), and supercomputers.

Linux File System Structure

Linux File System Structure

Linux में एक single tree structure होती है जो root (/) से शुरू होती है — Windows की तरह C:, D: drives नहीं होते।

Linux has a single tree structure starting from root (/) — unlike Windows, there are no C:, D: drives.

Linux File Permissions (chmod)

Linux File Permissions (chmod)

Linux में हर file के लिए 3 types की permissions होती हैं: r (read=4), w (write=2), x (execute=1) — तीन levels पर: Owner, Group, Others।

Every file in Linux has 3 types of permissions: r (read=4), w (write=2), x (execute=1) — at three levels: Owner, Group, Others.

chmod 755 file.txt # Owner: rwx(7), Group: r-x(5), Others: r-x(5)

Linux Distribution (Distro) क्या है?

What is a Linux Distribution (Distro)?

Linux Distribution Linux Kernel के ऊपर एक complete OS package है जिसमें GUI, applications, और tools शामिल होते हैं। अलग-अलग uses के लिए अलग-अलग distros available हैं।

A Linux Distribution is a complete OS package built on the Linux Kernel including a GUI, applications, and tools. Different distros are available for different uses.
