Unit 04 · एक्सेल बेसिक्सUnit 04 · Excel Basics

Cell Referencing

TypeExampleविशेषताFeature
Relative ReferenceA1Copy करने पर row/column automatically adjust होता हैRow/column automatically adjusts when copied
Absolute Reference$A$1Copy करने पर भी fix रहता है — $ दोनों से पहलेStays fixed when copied — $ before both
Mixed Reference$A1 या A$1सिर्फ column ($A1) या सिर्फ row (A$1) fixOnly column ($A1) or only row (A$1) fixed

Formula में cell reference select करके F4 press करने से Relative → Absolute → Mixed cycle में बदलता है।

In a formula, select the cell reference and press F4 to cycle through Relative → Absolute → Mixed.

याद रखें: Absolute reference ($A$1) तब ज़रूरी होता है जब formula copy करते समय एक specific cell (जैसे tax rate) को fix रखना हो।Remember: Absolute reference ($A$1) is needed when a specific cell (like tax rate) must stay fixed while copying a formula.

Unit 08 · FunctionsUnit 08 · Functions

Lookup Functions (VLOOKUP, HLOOKUP)

VLOOKUP

VLOOKUP

VLOOKUP (Vertical Lookup) table के पहले column में value ढूँढता है और उसी row से किसी और column की value return करता है।

VLOOKUP (Vertical Lookup) searches for a value in the first column of a table and returns a value from another column in the same row.

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

=VLOOKUP(A2,Sheet2!A:C,3,FALSE)
ArgumentमतलबMeaning
lookup_valueजो value ढूँढनी हैThe value to search for
table_arrayजिस range में ढूँढना हैThe range to search within
col_index_numकिस column से value लानी हैWhich column number to return value from
range_lookupFALSE = exact match, TRUE = approximateFALSE = exact match, TRUE = approximate

VLOOKUP vs HLOOKUP

VLOOKUP vs HLOOKUP

VLOOKUPHLOOKUP
Vertical — पहले column में ढूँढता हैVertical — searches in first columnHorizontal — पहली row में ढूँढता हैHorizontal — searches in first row
Exam Tip: VLOOKUP में lookup value हमेशा table_array के leftmost column में होनी चाहिए।Exam Tip: in VLOOKUP, the lookup value must always be in the leftmost column of the table_array.
