Data Communication परिचयData Communication Introduction

Data Communication क्या है?

What is Data Communication?

Data Communication दो devices के बीच data (information) को एक माध्यम (medium) से भेजने और प्राप्त करने की प्रक्रिया है। इसमें एक sender, एक receiver और बीच में एक transmission medium होता है।

Data Communication is the process of sending and receiving data (information) between two devices through a medium. It involves a sender, a receiver, and a transmission medium in between.

Data Communication के Components

Components of Data Communication

ComponentकामFunction
Messageवो information जो भेजी जा रही है (text, number, image, audio, video)The information being sent (text, number, image, audio, video)
Senderवो device जो message भेजता हैThe device that sends the message
Receiverवो device जो message प्राप्त करता हैThe device that receives the message
Mediumवो रास्ता जिससे data भेजा जाता है (cable, wireless)The path through which data travels (cable, wireless)
Protocolनियमों का सेट जो communication को control करता हैA set of rules that controls the communication

Bandwidth और Data Transfer Rate

Bandwidth and Data Transfer Rate

Bandwidth यह बताता है कि एक medium पर एक समय में कितना data transfer हो सकता है — इसे Hz (Hertz) या bps (bits per second) में मापा जाता है। ज़्यादा bandwidth मतलब ज़्यादा तेज़ data transfer।

Bandwidth tells how much data can be transferred over a medium at a time — measured in Hz (Hertz) or bps (bits per second). Higher bandwidth means faster data transfer.

याद रखें: Data Communication = Message + Sender + Receiver + Medium + Protocol। Remember: Data Communication = Message + Sender + Receiver + Medium + Protocol.

Transmission Modes (संचरण विधियाँ)Transmission Modes

Transmission Mode क्या है?

What is a Transmission Mode?

Transmission Mode बताता है कि दो devices के बीच data किस दिशा (direction) में भेजा जा सकता है।

Transmission Mode describes the direction in which data can be sent between two devices.

Transmission Modes के प्रकार

Types of Transmission Modes

ModeविशेषताFeatureउदाहरणExample
SimplexData सिर्फ एक दिशा में भेजा जाता हैData flows in only one directionKeyboard से Computer, TV BroadcastKeyboard to Computer, TV Broadcast
Half-Duplexदोनों दिशाओं में data जा सकता है, पर एक समय में सिर्फ एक दिशा मेंData can go both ways, but only one direction at a timeWalkie-TalkieWalkie-Talkie

याद रखने का तरीका: Simplex = एक तरफा रास्ता, Half-Duplex = दोनों तरफ पर बारी-बारी, Full-Duplex = दोनों तरफ एक साथ (जैसे फोन कॉल)। Memory tip: Simplex = one-way road, Half-Duplex = both ways but one at a time, Full-Duplex = both ways at once (like a phone call).

Switching Techniques (स्विचिंग तकनीक)Switching Techniques

Switching क्या है?

What is Switching?

Switching वो तकनीक है जिससे network में data को source से destination तक भेजने के लिए सही रास्ता चुना जाता है, खासकर जब कई devices एक ही network share कर रहे हों।

Switching is the technique used to choose the right path for sending data from source to destination in a network, especially when multiple devices share the same network.

Switching के प्रकार

Types of Switching

Switching TypeविशेषताFeature
Circuit SwitchingCommunication शुरू होने से पहले एक dedicated path बनाया जाता है जो पूरी बातचीत के लिए reserved रहता है — जैसे पुराना Telephone SystemA dedicated path is established before communication starts and stays reserved for the entire conversation — like old Telephone System
Packet SwitchingData को छोटे-छोटे packets में तोड़कर भेजा जाता है, हर packet अलग रास्ते से जा सकता है — Internet इसी पर काम करता हैData is broken into small packets and sent, each packet can take a different path — the Internet works on this
Message Switchingपूरा message एक साथ भेजा जाता है, हर node पर store होकर फिर आगे भेजा जाता है (Store and Forward)The whole message is sent together, stored at each node, then forwarded (Store and Forward)
याद रखें: Internet Packet Switching पर काम करता है — यही वजह है कि data efficiently और तेज़ी से भेजा जा सकता है। Remember: The Internet works on Packet Switching — that's why data can be sent efficiently and quickly.

Internet का परिचयIntroduction to Internet

Internet क्या है?

What is the Internet?

Internet दुनियाभर के लाखों-करोड़ों computers और networks को आपस में जोड़ने वाला सबसे बड़ा network है। इसे "Network of Networks" भी कहा जाता है। यह Packet Switching तकनीक पर काम करता है।

The Internet is the largest network connecting millions of computers and networks worldwide. It is also called the "Network of Networks". It works on Packet Switching technology.

Internet का इतिहास (संक्षेप में)

History of Internet (Brief)

Internet की शुरुआत 1960 के दशक में ARPANET (US Defence project) से हुई थी। 1990 में Tim Berners-Lee ने WWW (World Wide Web) बनाया, जिसके बाद Internet आम लोगों तक पहुंचना शुरू हुआ।

The Internet began in the 1960s with ARPANET (a US Defence project). In 1990, Tim Berners-Lee created the WWW (World Wide Web), after which the Internet started reaching common people.

Internet कैसे काम करता है?

How Does the Internet Work?

जब हम किसी website को खोलते हैं, तो हमारा computer (client) एक request, ISP और routers के through, उस website के server तक भेजता है। Server response को वापस उसी रास्ते से client तक भेजता है — यह सब seconds में होता है।

When we open a website, our computer (client) sends a request through the ISP and routers to that website's server. The server sends the response back to the client via the same path — all this happens in seconds.

URL और Domain Name

URL and Domain Name

URL (Uniform Resource Locator) किसी web page का पूरा पता होता है — जैसे https://www.google.com। Domain Name website का नाम होता है (google.com), जिसे लोग आसानी से याद रख सकें, जबकि असल में server का IP Address इस्तेमाल होता है।

A URL (Uniform Resource Locator) is the full address of a web page — like https://www.google.com. A Domain Name is the website's name (google.com), which people can easily remember, while the server actually uses an IP Address.

याद रखें: DNS (Domain Name System) Domain Name को IP Address में बदलता है ताकि Internet समझ सके कौन सा server access करना है। Remember: DNS (Domain Name System) converts a Domain Name into an IP Address so the Internet knows which server to access.

A Search Engine is a tool/website that searches through millions of pages on the Internet to find results matching the user's keywords. Examples — Google, Bing, Yahoo.

Email क्या है?

What is Email?

Email (Electronic Mail) Internet के through electronic messages भेजने और प्राप्त करने का तरीका है। यह तेज़, सस्ता और कागज़ की बचत करता है।

Email (Electronic Mail) is a way to send and receive electronic messages over the Internet. It is fast, cheap, and saves paper.

Structure of an Email Address

Example: suraj123@gmail.com — here suraj123 is the username, @ is the separator, and gmail.com is the domain name (the mail server's name).

  • Antivirus और Firewall चालू रखें
  • Public Wi-Fi पर banking/payment न करें

  • Data Communication — दो devices के बीच data भेजने-पाने की प्रक्रिया
  • Simplex — सिर्फ एक दिशा में data transfer
  • Full-Duplex — दोनों दिशाओं में एक साथ data transfer
  • Packet Switching — data को packets में तोड़कर अलग रास्तों से भेजना (Internet इसी पर चलता है)
  • Internet — दुनियाभर के networks को जोड़ने वाला सबसे बड़ा network
  • Intranet — एक organization का private network
  • URL — किसी web page का पूरा पता
  • ISP — Internet connection देने वाली कंपनी
  • Email — Internet के through electronic messages भेजना

  • Data Communication — the process of sending-receiving data between two devices
  • Simplex — data transfer in only one direction
  • Full-Duplex — data transfer in both directions at once
  • Packet Switching — breaking data into packets sent via different paths (the Internet runs on this)
  • Internet — the largest network connecting networks worldwide
  • Intranet — a private network of one organization
  • URL — the complete address of a web page
  • ISP — the company that provides Internet connection
  • Email — sending electronic messages over the Internet
  • E-Commerce — buying-selling online via the Internet
