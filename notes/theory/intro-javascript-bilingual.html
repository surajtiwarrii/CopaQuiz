JavaScript का परिचयIntroduction to JavaScript

JavaScript क्या है?

What is JavaScript?

JavaScript (JS) एक lightweight, interpreted programming language है जो webpages को interactive और dynamic बनाने के लिए use होती है। यह client-side (browser में) और server-side (Node.js से) दोनों जगह चल सकती है।

JavaScript (JS) is a lightweight, interpreted programming language used to make webpages interactive and dynamic. It can run both client-side (in the browser) and server-side (via Node.js).

HTML, CSS, JS का Role (Recap)

Role of HTML, CSS, JS (Recap)

TechnologyकामJob
HTMLStructure/Content (Skeleton)Structure/Content (Skeleton)
CSSStyle/Design (रंग, फॉन्ट, layout)Style/Design (colors, fonts, layout)
JavaScriptBehavior/Interactivity (clicks, validation, animation)Behavior/Interactivity (clicks, validation, animation)

JavaScript के उपयोग

Uses of JavaScript

  • Form validation (बिना page reload किए)
  • Buttons पर click events handle करना
  • Images/sliders animate करना
  • Web pages को बिना reload data update करना (AJAX)
  • Web apps, games, mobile apps बनाना
  • Form validation (without reloading the page)
  • Handling click events on buttons
  • Animating images/sliders
  • Updating page data without reload (AJAX)
  • Building web apps, games, mobile apps

Client-Side vs Server-Side JavaScript

Client-Side vs Server-Side JavaScript

Client-SideServer-Side
Browser में चलता है (jैसे form validation)Runs in the browser (e.g. form validation)Server पर चलता है (जैसे Node.js)Runs on the server (e.g. Node.js)
याद रखें: JavaScript और Java दो अलग languages हैं — नाम में similarity होने के अलावा इनमें कोई direct connection नहीं है। Remember: JavaScript and Java are two different languages — apart from the name similarity, there's no direct connection between them.

Variables & Data TypesVariables & Data Types

Variable क्या है?

What is a Variable?

Variable एक named container है जो data को store करता है, जिसे बाद में use या change किया जा सकता है।

A Variable is a named container that stores data, which can be used or changed later.

var, let, const में अंतर

Difference between var, let, const

KeywordविशेषताFeature
varपुराना तरीका, function-scoped, re-declare हो सकता हैOld way, function-scoped, can be re-declared
letBlock-scoped, value change हो सकती है पर re-declare नहींBlock-scoped, value can change but can't be re-declared
constBlock-scoped, value fix रहती है (constant)Block-scoped, value stays fixed (constant)

Variable Naming Rules

Variable Naming Rules

  • Letter, underscore (_) या $ से शुरू हो सकता है, number से नहीं
  • Spaces allowed नहीं हैं
  • Reserved keywords (जैसे let, function) use नहीं कर सकते
  • Case-sensitive होता है (age और Age अलग हैं)
  • Must start with a letter, underscore (_), or $, not a number
  • Spaces are not allowed
  • Cannot use reserved keywords (like let, function)
  • Case-sensitive (age and Age are different)
याद रखें: Modern JS में हमेशा let/const use करना चाहिए, var को avoid करना best practice है। Remember: in modern JS, always prefer let/const; avoiding var is best practice.

Unit 05 · कंट्रोल फ्लोUnit 05 · Control Flow

Conditional StatementsConditional Statements

if, else if, else

if, else if, else

Conditional statements code को decide करने देते हैं कि किस condition पर कौन सा block चलाना है।

Conditional statements let the code decide which block to run based on a condition.

let marks = 75; if (marks >= 90) { console.log("Grade A"); } else if (marks >= 60) { console.log("Grade B"); } else { console.log("Grade C"); }

switch Statement

switch Statement

जब एक variable की कई fixed values के against checking करनी हो, तब switch if-else की chain से ज़्यादा clean तरीका है।

When checking one variable against many fixed values, switch is a cleaner approach than a chain of if-else.

switch (day) { case 1: console.log("Monday"); break; case 2: console.log("Tuesday"); break; default: console.log("Other day"); }

Ternary Operator

Ternary Operator

यह if-else का shorthand है — single line में condition check करता है।

This is a shorthand for if-else — checks a condition in a single line.

let result = (age >= 18) ? "Adult" : "Minor";
याद रखें: switch में हर case के बाद break लगाना ज़रूरी है, वरना अगला case भी चल जाएगा (fall-through)। Remember: a break is required after each case in switch, otherwise the next case will also execute (fall-through).

LoopsLoops

Loop क्या है?

What is a Loop?

Loop किसी code block को बार-बार (repeatedly) चलाने के लिए use होता है, जब तक condition true रहती है।

A Loop is used to run a code block repeatedly, as long as a condition stays true.

for Loop

for Loop

for (let i = 1; i <= 5; i++) { console.log(i); }

for loop में 3 parts होते हैं: initialization, condition, increment/decrement — जब iterations की संख्या पहले से पता हो तब best है।

A for loop has 3 parts: initialization, condition, increment/decrement — best when the number of iterations is known beforehand.

while Loop

while Loop

let i = 1; while (i <= 5) { console.log(i); i++; }

while loop condition को पहले check करता है, फिर body run करता है — जब iterations की संख्या पहले से पता न हो तब useful है।

A while loop checks the condition first, then runs the body — useful when the number of iterations isn't known in advance.

do...while Loop

do...while Loop

let i = 1; do { console.log(i); i++; } while (i <= 5);

do...while body को पहले एक बार चलाता है, फिर condition check करता है — इसलिए body कम से कम एक बार ज़रूर चलती है।

do...while runs the body once first, then checks the condition — so the body always runs at least once.

break & continue

break & continue

KeywordकामPurpose
breakLoop को पूरी तरह रोक देता हैStops the loop completely
continueCurrent iteration skip करके अगली पर जाता हैSkips the current iteration and moves to the next
याद रखें: while loop में condition पहले check होती है (body 0 बार भी चल सकती है), do...while में body पहले run होती है (कम से कम 1 बार ज़रूर)। Remember: while checks the condition first (body may run 0 times), do...while runs the body first (always at least once).
संभावित परीक्षा प्रश्न Likely Exam Questions for loop का syntax उदाहरण सहित लिखिए। while और do...while में अंतर बताइए। break और continue में अंतर लिखिए। Write the syntax of a for loop with an example. Differentiate between while and do...while. Write the difference between break and continue.

FunctionsFunctions

Function क्या है?

What is a Function?

Function code का एक reusable block है जो एक specific task perform करता है। इसे एक बार define करके कई बार call किया जा सकता है।

A Function is a reusable block of code that performs a specific task. It's defined once and can be called multiple times.

Function Declaration & Calling

Function Declaration & Calling

function greet(name) { return "Hello, " + name; } console.log(greet("Suraj")); // calling the function

Parameters vs Arguments

Parameters vs Arguments

ParameterArgument
Function define करते समय दिए गए variablesVariables given when defining the functionFunction call करते समय दी गई actual valuesActual values given when calling the function

Function Expression & Arrow Function

Function Expression & Arrow Function

// Function Expression const add = function(a, b) { return a + b; }; // Arrow Function const add2 = (a, b) => a + b;

return Statement

return Statement

return function से एक value वापस भेजता है और function का execution वहीं रोक देता है। बिना return वाला function undefined return करता है।

return sends a value back from the function and stops the function's execution there. A function with no return statement returns undefined.

याद रखें: Function declaration hoisting के कारण file में कहीं भी define किया हो, ऊपर भी call हो सकता है — पर Arrow function/expression को define करने के बाद ही call करना पड़ता है। Remember: due to hoisting, a function declaration can be called even before it appears in the file — but arrow functions/expressions must be called only after they're defined.

Arrays & ObjectsArrays & Objects

Array क्या है?

What is an Array?

Array कई values को एक single variable में, ordered तरीके से store करता है। हर value का एक index होता है, जो 0 से शुरू होता है।

An Array stores multiple values in a single variable, in an ordered way. Each value has an index, starting from 0.

let fruits = ["Apple", "Banana", "Mango"]; console.log(fruits[0]); // "Apple" console.log(fruits.length); // 3

Common Array Methods

Common Array Methods

MethodकामPurpose
push()Array के end में नया element add करता हैAdds a new element to the end of the array
pop()Array के last element को remove करता हैRemoves the last element of the array
lengthArray में elements की संख्या बताता हैGives the number of elements in the array
indexOf()किसी value की index position ढूँढता हैFinds the index position of a value

Object क्या है?

What is an Object?

Object data को key: value pairs के रूप में store करता है — किसी real-world entity की properties को represent करने का natural तरीका है।

An Object stores data as key: value pairs — a natural way to represent properties of a real-world entity.

let student = { name: "Suraj", age: 21, course: "ITI-COPA" }; console.log(student.name); // "Suraj"

Array vs Object

Array vs Object

ArrayObject
Ordered list, index number से accessOrdered list, accessed via index numberKey-value pairs, key name से accessKey-value pairs, accessed via key name
[1, 2, 3]{a: 1, b: 2}
याद रखें: Array की indexing हमेशा 0 से शुरू होती है — पहला element index 0 पर होता है, दूसरा index 1 पर। Remember: Array indexing always starts at 0 — the first element is at index 0, the second at index 1.

Events & DOM बेसिक्सEvents & DOM Basics

DOM क्या है?

What is the DOM?

DOM (Document Object Model) HTML page को एक tree-structure की तरह represent करता है, जिससे JavaScript page के elements को access और change कर सकती है।

The DOM (Document Object Model) represents an HTML page as a tree structure, allowing JavaScript to access and change page elements.

DOM Methods

DOM Methods

MethodकामPurpose
getElementById()id से किसी element को select करता हैSelects an element by its id
querySelector()CSS selector से पहला matching element ढूँढता हैFinds the first matching element using a CSS selector
innerHTMLElement का content पढ़ता/बदलता हैReads/changes an element's content
document.getElementById("title").innerHTML = "New Title";

Event क्या है?

What is an Event?

Event कोई action है जो browser में होता है — जैसे click, mouse hover, key press, form submit। JS इन events को "listen" करके response दे सकती है।

An Event is an action that happens in the browser — like a click, mouse hover, key press, or form submit. JS can "listen" for these events and respond.

Common Event Types

Common Event Types

onchangeInput value बदलने परWhen input value changes
onmouseoverMouse element के ऊपर आने परWhen mouse hovers over the element
onloadPage पूरा load होने परWhen the page finishes loading
<button onclick="sayHi()">Click Me</button> <script> function sayHi() { alert("Hi there!"); } </script>
Exam Tip: DOM Manipulation का मतलब है JS के through HTML elements को add/remove/change करना — यह interactivity का core concept है। Exam Tip: DOM Manipulation means adding/removing/changing HTML elements via JS — it's the core concept behind interactivity.
