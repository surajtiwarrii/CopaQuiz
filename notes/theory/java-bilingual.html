Java का परिचयIntroduction to Java

Java क्या है?

What is Java?

Java एक high-level, object-oriented, platform-independent programming language है, जिसे Sun Microsystems ने 1995 में release किया था (अब Oracle द्वारा maintained)।

Java is a high-level, object-oriented, platform-independent programming language released by Sun Microsystems in 1995 (now maintained by Oracle).

Java की Key Features

Key Features of Java

FeatureमतलबMeaning
Platform Independent"Write Once, Run Anywhere" — कोई भी OS पर चलता है"Write Once, Run Anywhere" — runs on any OS
Object-OrientedClass और Object पर based programmingProgramming based on Class and Object
SimpleC++ जैसी complex चीज़ें (pointers) हटा दी गई हैंComplex features like pointers (from C++) are removed

RobustStrong memory management, exception handlingStrong memory management, exception handling
Multithreadedएक साथ कई tasks execute कर सकता हैCan execute multiple tasks simultaneously

JDK, JRE, JVMJDK, JRE, JVM

JDK, JRE, JVM में अंतर

Difference between JDK, JRE, JVM

TermमतलबMeaning
JVM
(Java Virtual Machine)(Java Virtual Machine)		Bytecode को run करने वाला engine, platform-specificThe engine that runs bytecode, platform-specific
JRE
(Java Runtime Environment)(Java Runtime Environment)		JVM + libraries, सिर्फ Java program चलाने के लिएJVM + libraries, used only to run Java programs
JDK
(Java Development Kit)(Java Development Kit)		JRE + compiler + development tools, programs बनाने और चलाने दोनों के लिएJRE + compiler + development tools, for both building and running programs

Relationship (JDK ⊃ JRE ⊃ JVM)

Relationship (JDK ⊃ JRE ⊃ JVM)

JDK सबसे बड़ा है — इसमें JRE शामिल है, और JRE में JVM शामिल है। Developer को JDK चाहिए, पर सिर्फ program चलाने वाले user को सिर्फ JRE काफी है।

JDK is the largest — it includes JRE, and JRE includes JVM. A developer needs JDK, but a user who just runs programs only needs JRE.

Java Program कैसे Run होता है?

How Does a Java Program Run?

  1. Source code (.java file) लिखा जाता है
  2. Compiler (javac) इसे Bytecode (.class file) में convert करता है
  3. JVM Bytecode को machine code में convert करके run करता है
  1. Source code (.java file) is written
  2. The compiler (javac) converts it into Bytecode (.class file)
  3. The JVM converts the Bytecode into machine code and runs it

Bytecode क्या है?

What is Bytecode?

Bytecode Java compiler द्वारा generate की गई intermediate code है — यही platform-independent होती है, और इसी वजह से Java "Write Once, Run Anywhere" बन पाती है।

Bytecode is the intermediate code generated by the Java compiler — it is platform-independent, which is what makes Java "Write Once, Run Anywhere".

याद रखें: JVM platform-dependent होता है (हर OS का अलग JVM), पर Bytecode platform-independent होता है — इसी combination से Java portable बनती है। Remember: the JVM is platform-dependent (each OS has its own JVM), but Bytecode is platform-independent — this combination makes Java portable.

Variables & Data TypesVariables & Data Types

Variable क्या है?

What is a Variable?

Variable memory का एक named location है जो data store करता है। Java में हर variable को एक data type के साथ declare करना ज़रूरी है (Statically Typed Language)।

A Variable is a named memory location that stores data. In Java, every variable must be declared with a data type (Statically Typed Language).

Control StatementsControl Statements

if, else if, else

if, else if, else

switch Statement

switch Statement

switch (day) { case 1: System.out.println("Monday"); break; case 2: System.out.println("Tuesday"); break; default: System.out.println("Other day"); }

हर case के बाद break लगाना ज़रूरी है, वरना अगला case भी execute हो जाएगा (fall-through)।

A break is required after each case, otherwise the next case also executes (fall-through).

Ternary Operator

Ternary Operator

याद रखें: switch सिर्फ int, char, String, और enum types के साथ काम करता है — float/double के साथ नहीं। Remember: switch only works with int, char, String, and enum types — not with float/double.

ArraysArrays

Array क्या है?

What is an Array?

Array एक same data type की कई values को एक continuous memory location में, fixed size के साथ store करता है। Index हमेशा 0 से शुरू होती है।

An Array stores multiple values of the same data type in a continuous memory location with a fixed size. Indexing always starts at 0.

Array Declaration & Initialization

Array Declaration & Initialization

int[] nums = new int[5]; int[] arr = {10, 20, 30, 40, 50}; System.out.println(arr[0]); // 10 System.out.println(arr.length); // 5

1D vs 2D Array

1D vs 2D Array

1D Array2D Array
Single row data, जैसे listSingle row of data, like a listRows और columns वाला data, जैसे matrix/tableData with rows and columns, like a matrix/table
int[] arr = {1,2,3};int[][] arr = {{1,2},{3,4}};

Array की विशेषताएं

Characteristics of Arrays

  • Fixed size — एक बार बनने के बाद size change नहीं हो सकता
  • Same data type की values ही store होती हैं
  • length property से size पता चलती है
  • Fixed size — cannot change size once created
  • Stores values of the same data type only
  • The length property gives the size
याद रखें: Array index हमेशा 0 से शुरू होती है — n size के array में आखिरी valid index n-1 होगी। Remember: array indexing always starts at 0 — in an array of size n, the last valid index is n-1.

Class, Object & MethodsClass, Object & Methods

Class क्या है?

What is a Class?

Class एक blueprint/template है जो objects की properties (variables) और behavior (methods) को define करता है।

A Class is a blueprint/template that defines an object's properties (variables) and behavior (methods).

Object क्या है?

What is an Object?

Object किसी Class का actual instance है — Class में defined properties और methods की real-world copy।

An Object is an actual instance of a Class — a real-world copy of the properties and methods defined in the Class.

class Student { String name; int age; void display() { System.out.println(name + " is " + age); } } // Creating an object Student s1 = new Student(); s1.name = "Suraj"; s1.age = 21; s1.display();

Constructor

Constructor

Constructor एक special method है जो object बनते ही automatically call होता है — initialization के लिए use होता है। इसका नाम class के नाम जैसा ही होता है, और कोई return type नहीं होता।

A Constructor is a special method automatically called when an object is created — used for initialization. It has the same name as the class and no return type.

class Student { String name; Student(String n) { // Constructor name = n; } }

Method Overloading

Method Overloading

एक ही class में same नाम के multiple methods, पर अलग parameters के साथ — Compile-time Polymorphism का उदाहरण है।

Multiple methods with the same name in one class, but with different parameters — an example of Compile-time Polymorphism.

याद रखें: एक class से कई objects बनाए जा सकते हैं, और हर object की अपनी अलग copy होती है instance variables की। Remember: multiple objects can be created from one class, and each object has its own separate copy of instance variables.

Inheritance Example

Inheritance Example

class Animal { void eat() { System.out.println("Eating..."); } } class Dog extends Animal { void bark() { System.out.println("Barking..."); } } // Dog object can use both eat() and bark()

Types of Inheritance

Java multiple inheritance (एक class का दो classes से inherit) को classes के साथ support नहीं करता — सिर्फ interfaces के through मुमकिन है।

Java does not support multiple inheritance with classes (one class inheriting from two) — it's only possible through interfaces.

Encapsulation Example

Encapsulation Example

class Account { private double balance; // data hidden public double getBalance() { return balance; } }

private keyword से variables को directly बाहर से access करने से रोका जाता है — सिर्फ getter/setter methods से access मिलता है।

The private keyword prevents direct outside access to variables — access is given only via getter/setter methods.

Exam Tip: Method Overloading Compile-time Polymorphism है, Method Overriding Run-time Polymorphism — exam में अक्सर इनका अंतर पूछा जाता है। Exam Tip: Method Overloading is Compile-time Polymorphism, Method Overriding is Run-time Polymorphism — their difference is frequently asked in exams.

Exception HandlingException Handling

Exception क्या है?

What is an Exception?

Exception program execution के दौरान आने वाली एक unwanted/unexpected event है, जो program के normal flow को disrupt करती है (जैसे divide by zero)।

An Exception is an unwanted/unexpected event during program execution that disrupts the normal flow of the program (like divide by zero).

try, catch, finally

try, catch, finally
