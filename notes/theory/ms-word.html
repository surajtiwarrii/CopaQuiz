MS Word क्या है?

What is MS Word?

Microsoft Word Microsoft Office Suite का एक Word Processor application है जो professional documents जैसे letters, reports, resumes, books बनाने के लिए use होता है।

Microsoft Word is a Word Processor application in the Microsoft Office Suite, used to create professional documents like letters, reports, resumes, and books.

Line Spacing & Paragraph Spacing

Line Spacing & Paragraph Spacing

Line Spacing lines के बीच की vertical space है — Single, 1.5, Double, Multiple options होते हैं। Paragraph Spacing paragraphs के ऊपर और नीचे की space है — Before और After में set होती है।

Line Spacing is the vertical space between lines — options include Single, 1.5, Double, Multiple. Paragraph Spacing is the space above and below paragraphs — set via Before and After.

Header & Footer

Header & Footer

Header हर page के top पर repeat होने वाला text/image area है (जैसे company name, logo)। Footer हर page के bottom पर repeat होने वाला area है (जैसे page number, copyright)।

A Header is an area at the top of every page for repeating text/image (like company name, logo). A Footer is an area at the bottom of every page (like page number, copyright).

Double-click करने से Header/Footer edit mode में जाते हैं। Header/Footer में जाने के बाद main document edit नहीं होता।

Double-clicking enters Header/Footer edit mode. While editing a Header/Footer, the main document cannot be edited.

Page Numbers

Page Numbers

Insert → Page Number से pages पर automatic numbering add होती है — Top of Page, Bottom of Page, Current Position options।

Insert → Page Number adds automatic numbering to pages — options include Top of Page, Bottom of Page, Current Position.

Mail Merge

Mail Merge क्या है?

What is Mail Merge?

Mail Merge एक main document (letter template) को एक data source (list of names/addresses) से combine करके automatically कई personalized letters/labels बनाने की facility है।

Mail Merge is a feature that automatically combines a main document (letter template) with a data source (list of names/addresses) to create many personalised letters/labels.

Mail Merge के लिए क्या चाहिए?

What is needed for Mail Merge?

ComponentमतलबMeaning
Main DocumentLetter template जिसमें placeholders (merge fields) होते हैंLetter template containing placeholders (merge fields)
Data SourceNames, addresses की list — Excel sheet, Word table, या databaseList of names, addresses — Excel sheet, Word table, or database
Merged DocumentFinal output — हर record के लिए अलग letterFinal output — a separate letter for each record

Mail Merge के Steps

Steps of Mail Merge

  1. Mailings Tab → Start Mail Merge → Letters
  2. Select Recipients → Use Existing List (Excel file) या Type New List
  3. Main document में merge fields insert करें (जैसे <<FirstName>>)
  4. Preview Results — देखें कि letter कैसा दिखेगा
  5. Finish & Merge → Print Documents या Edit Individual Documents
  1. Mailings Tab → Start Mail Merge → Letters
  2. Select Recipients → Use Existing List (Excel file) or Type New List
  3. Insert merge fields in the main document (like <<FirstName>>)
  4. Preview Results — see what the letter will look like
  5. Finish & Merge → Print Documents or Edit Individual Documents
Exam Tip: Mail Merge को "Bulk Mailing" भी कहते हैं — इसका use exam results, invitation letters, certificates bulk में send करने के लिए होता है।Exam Tip: Mail Merge is also called "Bulk Mailing" — it is used for sending exam results, invitation letters, certificates in bulk.

References, Track Changes & CommentsReferences, Track Changes & Comments

Table of Contents (TOC)

Table of Contents (TOC)

References Tab → Table of Contents — Headings के आधार पर automatic TOC बनता है जो document की शुरुआत में chapters और page numbers दिखाता है। Headings को Heading 1, Heading 2 styles apply करने के बाद TOC generate होती है।

References Tab → Table of Contents — automatically creates a TOC based on Headings, showing chapters and page numbers at the start of the document. Apply Heading 1, Heading 2 styles to headings first, then generate the TOC.

Track Changes

Track Changes

Track Changes (Ctrl+Shift+E) document में किए गए सभी changes (additions, deletions) को mark करके record करता है — team collaboration में हर author के changes अलग color में दिखते हैं।

Track Changes (Ctrl+Shift+E) records all changes made to the document (additions, deletions) — in team collaboration, each author's changes appear in a different colour.

Comments

Comments

Review Tab → New Comment (Ctrl+Alt+M) — document के किसी part पर note/feedback add किया जा सकता है बिना main content change किए। Reply और Resolve options होते हैं।

Review Tab → New Comment (Ctrl+Alt+M) — a note/feedback can be added to any part of the document without changing the main content. Reply and Resolve options are available.
