Microsoft PowerPoint Microsoft Office का Presentation software है जो visual slides-based presentations बनाने के लिए use होता है — meetings, lectures, seminars, और demos के लिए।

Microsoft PowerPoint is the Presentation software in Microsoft Office, used to create visual slides-based presentations — for meetings, lectures, seminars, and demos.

Slide Layouts & Themes/DesignSlide Layouts & Themes/Design

Slide Layout क्या है?

What is Slide Layout?

Slide Layout slide पर content (title, text, images) की pre-arranged structure है — Title Slide, Title and Content, Blank, Two Content जैसे layouts होते हैं।

A Slide Layout is a pre-arranged structure for content (title, text, images) on a slide — layouts include Title Slide, Title and Content, Blank, Two Content etc.

LayoutमतलबMeaning
Title SlidePresentation का first slide — big title + subtitleFirst slide of presentation — big title + subtitle
Title and ContentTitle + content area (text, image, chart)Title + content area (text, image, chart)
Two ContentTitle + दो side-by-side content areasTitle + two side-by-side content areas
BlankEmpty slide — manually सब add करनाEmpty slide — add everything manually

Themes & Design

Themes & Design

Theme colors, fonts, और effects का एक pre-designed combination है जो पूरी presentation पर apply होता है — Design Tab → Themes।

A Theme is a pre-designed combination of colours, fonts, and effects applied to the entire presentation — Design Tab → Themes.

OptionकामPurpose
ThemesPre-built professional color + font combinationsPre-built professional colour + font combinations
VariantsSame theme के different color variationsDifferent colour variations of the same theme
Background StylesSlide background को solid, gradient, texture से change करनाChanging slide background to solid, gradient, texture
Slide SizeStandard (4:3) या Widescreen (16:9) set करनाSetting Standard (4:3) or Widescreen (16:9)
याद रखें: Theme पूरी presentation पर लागू होती है, जबकि Layout सिर्फ एक slide का structure define करता है।Remember: a Theme applies to the entire presentation, while a Layout defines the structure of just one slide.

Text Boxes & Placeholders

Text Boxes & Placeholders

Slide पर text add करने के दो तरीके: (1) Placeholder में click करके type करें, (2) Insert Tab → Text Box draw करके type करें।

Two ways to add text to a slide: (1) Click inside a Placeholder and type, (2) Insert Tab → draw a Text Box and type.

Bullet Points & Numbered Lists

Bullet Points & Numbered Lists

Home Tab → Bullets (Ctrl+Shift+L) या Numbering से list format apply करें। Tab key से indent/level बढ़ाया जाता है, Shift+Tab से घटाया जाता है।

Apply list format via Home Tab → Bullets (Ctrl+Shift+L) or Numbering. Use the Tab key to increase indent/level, Shift+Tab to decrease it.

WordArt

WordArt

Animations

Animation क्या है?

What is Animation?

Animation slide पर किसी object (text, image, shape) के appear होने, disappear होने, या move होने का visual effect है — Animations Tab से apply होता है।

Animation is a visual effect for how an object (text, image, shape) appears, disappears, or moves on a slide — applied from the Animations Tab.

Animation Categories

Animation Categories

CategoryकामPurpose
EntranceObject slide पर कैसे appear होगा — Appear, Fade, Fly In, BounceHow the object appears on the slide — Appear, Fade, Fly In, Bounce
EmphasisObject already visible है, उसे highlight करना — Pulse, Spin, Grow/ShrinkHighlighting an already visible object — Pulse, Spin, Grow/Shrink
ExitObject slide से कैसे disappear होगा — Fade, Fly Out, ShrinkHow the object disappears from the slide — Fade, Fly Out, Shrink
Motion PathsObject एक specific path पर move करेगाThe object moves along a specific path

Animation Timing Options

Animation Timing Options

OptionमतलबMeaning
On ClickMouse click पर animation start होगीAnimation starts on mouse click
With Previousपिछली animation के साथ simultaneously startStarts simultaneously with the previous animation
After Previousपिछली animation खत्म होने के बाद automatically startStarts automatically after the previous animation ends
याद रखें: Animation किसी एक slide पर objects के effects हैं, Transition दो slides के बीच का effect है — दोनों अलग concepts हैं।Remember: Animation is an effect on objects within one slide, Transition is an effect between two slides — they are different concepts.

Transitions & Slide Show

Transition क्या है?

What is a Transition?

Transition एक slide से दूसरी slide पर switch होने का visual effect है — Transitions Tab से apply होता है।

A Transition is the visual effect when switching from one slide to another — applied from the Transitions Tab.

Transition TypeExamplesExamples
SubtleFade, Push, Wipe, Split
ExcitingFlip, Cube, Gallery, Dissolve
Dynamic ContentPan, Ferris Wheel, Conveyor

Transition Settings

Transition Settings

  • Duration — Transition कितनी slow/fast हो
  • Sound — Transition पर sound effect
  • Advance Slide: On Mouse Click या Automatically After X seconds
  • "Apply to All" — सभी slides पर same transition
  • Duration — how slow/fast the transition is
  • Sound — sound effect on transition
  • Advance Slide: On Mouse Click or Automatically After X seconds
  • "Apply to All" — same transition on all slides

Slide Master, Printing & HandoutsSlide Master, Printing & Handouts

Slide Master

Slide Master

Slide Master एक master template है जहाँ logo, font style, background जैसी चीज़ें एक बार set करने पर पूरी presentation के सभी slides पर automatically apply हो जाती हैं।

A Slide Master is a master template where setting things like logo, font style, background once automatically applies them to all slides in the presentation.

View Tab → Slide Master — यहाँ editing करने के बाद "Close Master View" से normal mode में वापस जाएं।

View Tab → Slide Master — after editing here, click "Close Master View" to return to normal mode.

Printing Options

Printing Options

Print LayoutमतलबMeaning
Full Page Slidesएक page पर एक slideOne slide per page
Notes PagesSlide + Speaker notes साथ में printSlide + Speaker notes printed together
Outlineसिर्फ text content printPrint text content only
Handouts (1,2,3,4,6,9 slides/page)एक page पर multiple slides — audience को देने के लिएMultiple slides per page — to give to audience

Speaker Notes

Speaker Notes

Speaker Notes slide के नीचे presenter के personal notes हैं — audience को नहीं दिखते, presenter को Presenter View में दिखते हैं।

Speaker Notes are the presenter's personal notes below the slide — not visible to the audience, visible to the presenter in Presenter View.

याद रखें: Handouts audience को distribute किए जाते हैं ताकि वे notes ले सकें, जबकि Speaker Notes सिर्फ presenter के लिए होते हैं।Remember: Handouts are distributed to the audience for note-taking, while Speaker Notes are only for the presenter.

1-line Definitions

1-line Definitions

  • MS PowerPoint — slides-based visual presentations बनाने का Microsoft application
  • Slide Layout — slide पर content का pre-arranged structure
  • Theme — colors, fonts, effects का combination जो पूरी presentation पर apply होता है

  • Handouts — audience को distribute करने के लिए multiple slides per page printout
